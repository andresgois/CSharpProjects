
# DotNet

## Comandos CLI
> Versão do DotNet
- dotnet --version
> Projeto tipo console
- dotnet new console
> Executa o código
- dotnet run
- dotnet watch run
> Cria API WEB
- dotnet new web
    - Muda porta da aplicação

## NuGet
- Centralizador de pacotes
[NuGet](https://www.nuget.org/)
> Pacote que altera a cor do terminal
- dotnet add package FluentColorConsole --version 2.0.1
